Unit uc_animate

DescriptionUsesClasses, Interfaces, Objects and RecordsFunctions and ProceduresTypesConstantsVariables

Description

This Unit has Some Console animations, some resembling ANSI ART. These are fun, good code examples of the uc_console.pp unit. I use this as a screen saver in the console version of JAS/JegasCRM's setp program.

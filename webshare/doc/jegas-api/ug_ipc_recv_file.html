Description

File Based Interprocess Communication

FileName: YYYYMMDDHHNNSSCCCRRRRRRRRRR.extension YYYY = YEAR CCC = Milli Seconds MM = Month RRRRRRRRRR = RANDOM NUMBER (31 bit range) DD = DAY HH = HOUR NN = MINUTES SS = SECONDS

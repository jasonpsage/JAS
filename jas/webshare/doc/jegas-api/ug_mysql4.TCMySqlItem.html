Function ExecuteQuery(p_sa: AnsiString):Boolean;

Executes query using THIS connection calls mysql_store_result internally - don't need to call it yourself. however - when you are through with the query - call DoneQuery so mysql_free_result gets called.

Function ExecuteQueryFromFile(p_sa: AnsiString):Boolean;

Works Just Like Execute Query except it executes the specified file if it can find it and load it.

Function HaveResults: Boolean;

returns true if current connection MySQLXDL is pointing to. This only works if you follow recommendations - see "QResult" above.

Function SelectDatabase(p_sa:AnsiString):Boolean;

Calls mysql_select_db for current connection MySqlXDL is pointing to.

Procedure DoneQuery;

Calls mysql_free_result for current connection MYSqlXDL is pointing to.

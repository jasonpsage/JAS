function saXML( p_bStripComments: boolean; p_bStripWhiteSpace: boolean; p_bRenderThisNodeandChildrenOnly: boolean; i4Nest: longint ):ansistring;

Renders SPECIFIC (current) NODE or all data loaded in the JFC_XML object. Generally you pass a ONE to the i4Nest parameter. It effects whitespace indentation fo human reading of the XML. Eliminate the possibility of this function rendering whitespace, pass a negative ONE.
